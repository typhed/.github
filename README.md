This is a special 🌟 repository that contains *community health files* for [`@TyPhed`](https://github.com/typhed) that are
global to the organization (e.g., `CODE_OF_CONDUCT.md`, `CONTRIBUTING.md`, etc.). A local file always overrides the global
configuration. Contents of the `README.MD` file is for information purposes, and hosts links, resources to help endusers. All
the issue templates, funding informations etc. are controlled from this directory.
